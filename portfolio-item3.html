L'application tablette de Pocramé

La santé dans les zones isolées

Pocramé est une startup née en 2014 qui a pour but d'améliorer la détection de maladie dans les milieux isolés tel que les bateaux ou zones rurales.
Pocramé offre deux services. Celui pour lequel j'ai travaillé est un lab permettant d'effectuer des tests de depistage.

Le laboratoire est accompagné d'une application mobile spécialement pour tablette. J'ai recréé l'application pour qu'elle soit adaptée aux tablettes utilisées et à leur système d'opération : Android. Je l'ai donc développé en Java.

L'application est composé de 3 parties. Une partie vidéo/tutoriel pour savoir comment effectuer les tests. Une partie historique permettant de faire un suivi des tests effectués. Une partie test où, en choisissant les différents symptomes, une liste de test à effectuer apparaît.
